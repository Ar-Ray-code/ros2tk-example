# ros2tk-example
ROS2 Python + tkinter

![](images_for_readme/pubint_example.gif)

## Demo

```
git clone https://github.com/Ar-Ray-code/ros2tk-example.git
cd ros2tk-example/

source /opt/ros/foxy/setup.bash
python3 pubint-tk.py
```

If you want to check the topic, run the following command.

```
source /opt/ros/foxy/setup.bash
ros2 topic echo /int
```
